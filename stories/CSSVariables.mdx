# CSS Variables

`@aid-on/design-system/css` を import することで、`:root[data-theme="light"]` / `:root[data-theme="dark"]` 配下に CSS カスタムプロパティとして展開されます。

## 一覧（Light）

<table className="aid-table">
  <thead>
    <tr>
      <th style={{ width: '30%' }}>変数名</th>
      <th style={{ width: '20%' }}>値</th>
      <th>説明</th>
    </tr>
  </thead>
  <tbody>
    {getAllColorTokens('light').map((t, i) => (
      <tr key={`light-${i}-${t.name}`}>
        <td><code>{t.name}</code></td>
        <td>
          <span style={{ display: 'inline-block', width: 14, height: 14, background: t.value, border: '1px solid #ddd', verticalAlign: 'middle', marginRight: 6, borderRadius: 3 }} />
          <code>{t.value}</code>
        </td>
        <td>{t.description}</td>
      </tr>
    ))}
  </tbody>
</table>
